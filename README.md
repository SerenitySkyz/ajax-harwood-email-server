Ajax Harwood Clinic — Voice Agent & Email Notification Server
This repository contains the webhook infrastructure and voice agent system prompt developed by Ajax Web AI for Ajax Harwood Medical Clinic.
About This Project
Ajax Web AI designed, built, and deployed a complete AI-powered voice agent solution for a medical clinic environment. This included:

A voice agent system prompt engineered specifically for clinical administrative use — including interruption handling, hesitation reduction, disclosure rules, data collection logic, safety guardrails, and smooth handoff behaviour
A Node.js / Express webhook server to handle Retell AI call event notifications and route email summaries to clinic staff via Resend
Deployment via Render, with the repository hosted under GitHub user SerenitySkyz
